# useActivePeers

The  ` useActivePeers `  hook returns the last most active "N" number of peers i.e. `activePeerIds` & `dominantSpeakerId`.


```tsx
const {
    activePeerIds,
    dominantSpeakerId,
  } = useActivePeers();
```

## Returns
The useActivePeers hook returns an object with the following fields.
