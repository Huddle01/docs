# Remote Peer

## Methods

### StartConsumerPeer(string peerId);

Call this function to consume individual peer. This function will invoke Event [`Huddle01Init.OnStartConsumingPeer`](/UnityWebglPlugin/methods/remotePeer#startconsumingpeereventhandlerstring-peerid) once the process is successfully done

<OptionTable
  headers={["Name", "Type", "Description"]}
  options={[["peerId", "string", "remote peer id"]]}
/>

```csharp filename=".csharp"
public void ConsumePeer(string peerId)
{
    Huddle01Init.Instance.StartConsumerPeer(peerId);
}
```

### StopConsumePeer(string peerId,string label);

Call this function to stop consuming individual peer.This function will invoke Event [`Huddle01Init.OnStopConsumingPeer`](/UnityWebglPlugin/methods/remotePeer#stopconsumingpeereventhandlerstring-peerid) once the process is successfully done

```csharp filename=".csharp"
private void OnEnable()
{
    Huddle01Init.PeerAdded += OnPeerJoined;
    Huddle01Init.PeerLeft += OnPeerLeft;
    Huddle01Init.PeerMuted += OnPeerMuted;
    Huddle01Init.PeerMetadata += OnPeerMetaDataUpdated;
    Huddle01Init.OnResumePeerVideo += OnPeerVideoResume;
    Huddle01Init.OnStopPeerVideo += OnPeerVideoStop;
    Huddle01Init.OnStartConsumingPeer += OnPeerVideoStop;
    Huddle01Init.OnStopConsumingPeer += OnPeerVideoStop;
}

private void OnDisable()
{
    Huddle01Init.PeerAdded -= OnPeerJoined;
    Huddle01Init.PeerLeft -= OnPeerLeft;
    Huddle01Init.PeerMuted -= OnPeerMuted;
    Huddle01Init.PeerMetadata -= OnPeerMetaDataUpdated;
    Huddle01Init.OnResumePeerVideo -= OnPeerVideoResume;
    Huddle01Init.OnStopPeerVideo -= OnPeerVideoStop;
    Huddle01Init.OnStartConsumingPeer -= OnPeerVideoStop;
    Huddle01Init.OnStopConsumingPeer -= OnPeerVideoStop;
}
```
