   <TabStrip version="0.0.17" date="4 Oct 2023"> 
    `@huddle01/web-core@0.0.17`
  - Admit/Deny peers
  ```tsx
   // Admit Peers
  huddleClient.admitPeer(["peerId-1", "peerId-2"]);
  huddleClient.denyPeer(["peerId-1", "peerId-2"]);
  ```
  - Lock/Unlock room 
  ```tsx
  // Lock Room 
  huddleClient.changeRoomControls("roomLocked", true);
  // Unlock Room
  huddleClient.changeRoomControls("roomLocked", false);
  ```
  - Get `lobbyPeers` from `huddleClient` which are waiting in lobby to join the room
  ```tsx
  // Get lobbyPeers
  const lobbyPeers = huddleClient.lobbyPeers;
  ```
  - Screen Share 
  ```tsx
  // Request Screen from browser
  huddleClient.fetchScreenShare()

  // Produce Screen Share to others
  huddleClient.produceScreenShare()

  // Stop Producing Screen Share to others
  huddleClient.stopProducingScreenShare()

  // Stop Screen Share Locally 
  huddleClient.stopScreenShare()
  ```
   </TabStrip>    

   <TabStrip version="0.0.16" date="20 Sept 2023">

  - The `previewPeers` array can be accessed from `huddleClient`.
  ```tsx
   // Get previewPeers
  const previewPeers = huddleClient.previewPeers;
  ```
   </TabStrip>    

  <TabStrip version="0.0.15" date="26 July 2023"> 
    `@huddle01/web-core@0.0.15`
  - Features
  - `sendData` func can be accessed from `huddleClient`
  ```tsx
     // Send data to all peers
     huddleClient.sendData("*", anyData);

     // Send data to multiple peers
     huddleClient.sendData(["peerId1", "peerId2"], anyData); 
  ```
  `getMeId` can be accessed from `huddleClient`

  ```tsx
     const roomId = huddleClient.getRoomId(); 
  ```
   </TabStrip>    


   <TabStrip version="0.0.14" date="5 July 2023"> 
    `@huddle01/web-core@0.0.14`

    - `app:cam-on` instead of `lobby:cam-on`
    - `useMeetingMachine()` deprecated, `roomState` from `useHuddle01()`
    - `fetchAudioStream()` and `fetchVideoStream()` can be called
    - `useEventListener()` for events `app:cam-on` and `app:mic-on` , `stream` is received in callback params.

   ```tsx
       useEventListener("app:cam-on", (stream: MediaStream) => {
		      console.log({stream});
        }); 
  ```
   </TabStrip>    


   <TabStrip version="0.0.13" date="19 April 2023"> 
    `@huddle01/web-core@0.0.13`

   - Added `useDisplayName()` hook which export 
   - `setDisplayName(displayName: string)` method which accepts a Display Name of data-type `string`
   - `displayName` - current Display Name of the user 
   - `isLoading` - loading state while displayName is getting updated
   - `error` - error if there was an error setting displayName
   - Exported `useLivestream()` and `useRecording()` hooks from `@huddle01/react/hooks` 
   - Added optional parameter accessToken to  joinLobby() method for token gating.
   - `joinLobby(roomId: string, accessToken: string)`
   - Added `room:audio-producing` and `room:video-producing` events
   - Added `useRecorder()` hook to allow recorder to join.
   - `roomId` is the first mandatory param
   - `projectId` is the second mandatory param

   </TabStrip>    
   </Tab>
