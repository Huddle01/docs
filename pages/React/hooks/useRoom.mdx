# useRoom
The useRoom hook exposes primitives to interact with a Huddle01 room, like joining, leaving, closing the room.
```tsx
const {
    room,
    state,
    joinRoom,
    leaveRoom,
    closeRoom,
    kickPeer,
    muteEveryone,
    closeStreamOfLabel,
  } = useRoom({
  onJoin(data:Room) ();
  onWaiting(data: RoomEvents['room-waiting'][0]) ();
  onLeave(data: RoomEvents['room-closed'][0]) ();
  onFailed(data: RoomEvents['room-joined-failed'][0]) ();
  onPeerJoin(peerId: string) ();
  onPeerLeft(peerId: string) ();
  });
```
## Props
The useRoom hook accepts an object with the following fields as props.
