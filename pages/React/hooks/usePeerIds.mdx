# usePeerIds

The usePeerIds hook returns peerIds of all peers inside a Huddle01 room.

```tsx
const {
    peerIds
  } = usePeerIds({
   roles?: string[];
  labels?: Array<
    'audio' | 'video' | 'screen-share-video' | 'screen-share-audio' | string
  >;
  });
```

## Props
The usePeerIds hook accepts an object with the following fields as props.
