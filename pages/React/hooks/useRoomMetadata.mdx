# useRoomMetadata

The  ` useRoomMetadata `  hook returns the data of the room and the function to update it`.


```tsx
const {
    roomData,
    updateMetadata,
  } = useRoomMetadata({
    onMetadataUpdated(metadata:unknown) {},
    onLeave(data: RoomEvents['room-closed'][0]) {}
  });
```

## Props
The useRoomMetadata hook accepts an object with the following fields as props.

```tsx

    console.log("Triggered when metadata updated")
```
