# useActivePeers

The `useActivePeers` hook returns the last most active "N" number of peers i.e. `activePeerIds` & `dominantSpeakerId`.

<OptionTable
  headers={['Description', 'Type']}
  options={[
    [
      'last most active n number of peers.',
      'string[]',
    ],
    [
      'last most active peer.',
      'string',
    ],
  ]}
/>

#### Example Usage 

```tsx
const {
    activePeerIds,
    dominantSpeakerId,
  } = useActivePeers();
```
