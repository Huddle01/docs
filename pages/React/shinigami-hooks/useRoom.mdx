# useRoom

The useRoom hook exposes primitives to interact with a Huddle01 room, like joining, leaving, closing the room.

```tsx
const {
    room,
    state,
    joinRoom,
    leaveRoom,
    closeRoom,
  } = useRoom({
    onJoin(data) {},
    onClose() {},
    onLeave() {},
  });
```


## Props
The useRoom hook accepts an object with the following fields as props.
