## Quickstart

### Installation

To get started with the Huddle01 React SDK, you can install the package using npm or yarn:

<Tabs items={['pnpm','npm', 'yarn']}>
 <Tab>

    ```bash
    pnpm install @huddle01/react@alpha
     ```
     </Tab>

  <Tab>
    
    ```bash 
    npm install @huddle01/react@alpha
     ```

  </Tab>

  <Tab>

    ```bash
    yarn add @huddle01/react@alpha
     ```
     </Tab>

</Tabs>

## Initializing the SDK:

First you need to configure HuddleProvider in your root component. 

```jsx
import { HuddleProvider, HuddleClient } from '@huddle01/react';

const huddleClient = new HuddleClient({
  projectId: env.NEXT_PUBLIC_PROJECT_ID,
  options: {
    activeSpeakers: {
      size: 14,
    },
  },
});

...

<HuddleProvider client={huddleCLient}>
  <App />
</HuddleProvider>
...
```

### Joining Room 

```jsx
import { useRoom } from '@huddle01/react/hooks';

const { joinRoom } = useRoom({
   onJoin: () => {
     console.log('Joined room');
   }
});

// join room 
joinRoom({
   roomId: "YOUR_ROOM_ID",
   token: "YOUR_TOKEN"
});
```
