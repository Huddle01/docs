# Get Participants 

This method returns a list of participants in a meetings. By passing the `meetingId`, you can 
easily access a list of participants who joined the meeting. 

<Callout>
You can retrieve the `meetingId` by using `getRoomMeetings()`. 
</Callout>

### Example 

```ts 
import { API } from '@huddle01/server-sdk/api';

const participantsList = async () => {
  const api = new API({
    apiKey: process.env.API_KEY!,
  });

  const participants = await api.getParticipants({
    meetingId: 'YOUR_MEETING_ID',
  });

  return participants?.data;
};
```
