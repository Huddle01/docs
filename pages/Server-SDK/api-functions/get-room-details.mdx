# Get Room Details 

`getRoomDetails()` method helps to retrieve information about specific `roomId`. 

```ts 
import { API } from '@huddle01/server-sdk/api';

const roomDetails = async () => {
  const api = new API({
    apiKey: process.env.API_KEY!,
  });

  const roomDetails = await api.getRoomDetails({
    roomId: 'YOUR_ROOM_ID',
  });

  return roomDetails?.data;
};
```

### Returns 

`getRoomDetails()` will return all details about room, if it's token gated it will 
also return `tokenGatingInfo`.
