# Get Live Meetings 

This method retrieves the list of live meetings from a given `API_KEY`. 
Live meetings are meetings where we have more than and equal to 1 participant.

### Example 

```ts 
import { API } from '@huddle01/server-sdk/api';

const liveMeetings = async () => {
  const api = new API({
    apiKey: process.env.API_KEY!,
  });

  const liveMeetings = await api.getLiveMeetings();

  return liveMeetings.data?.liveMeetings;
};
```
