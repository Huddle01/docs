# Create Room

`createRoom()` will create a new meeting room and return the room ID. This room ID can be used to join the room using `joinRoom()` method on client side.
While creating room, you can pass following details. 

## Parameters 

<CreateRoom />

## Example

```ts
import { API } from '@huddle01/server-sdk/api';

const getRoomId = async () => {
  const api = new API({
    apiKey: process.env.API_KEY!,
  });

  const createNewRoom = await api.createRoom({
    roomLocked: true,
    metadata: JSON.stringify({
      'title': 'Huddle01 Meeting',
    })
  });

  const roomId = createNewRoom?.data;

  return roomId;
};
```
