# Get Rooms 

`getRooms()` method will return all the rooms created from a given `API_KEY`. 

```ts 
import { API } from '@huddle01/server-sdk/api';

const rooms = async () => {
  const api = new API({
    apiKey: process.env.API_KEY!,
  });

  const rooms = await api.getRooms();

  console.log(rooms?.data);

  return rooms?.data;
};
```
