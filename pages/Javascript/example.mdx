## Clone Example App

### Using git

Run the following command in your terminal to download the example:

```bash 
  git clone https://github.com/Huddle01/vanilla-js-example
```

### Change directory
```bash 
  cd vanilla-js-example
```
### Serve the app

```bash 
  serve -s . -l 3000
```
