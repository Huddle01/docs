## Video 

### Enable Screen Share 

The `enableScreenShare` method enables the local screen sharing and starts producing the screen sharing stream with the label `screen`.

```js
// Enable screen share
await huddleClient.localPeer.enableScreenShare();
```

### Disable Screen Share

The `disableScreenShare` method stops the local screen sharing and also stops producing the screen sharing stream with remote peers.

```js
// Disable screen share
await huddleClient.localPeer.disableScreenShare();
```
