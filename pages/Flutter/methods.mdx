## Using client to access Methods


Once done, you can initialize your project with the `HuddleClient` class by passing in your `projectId` and access various methods such as `joinRoom` and other methods. 
 
<Callout>
You can generate Access Token using our [Server SDK](/Server-SDK). To generate `roomId` you can use our [Create Room API](/apis/create-room/create-room).
</Callout>

  ```dart filename="app.dart"
  import 'package:huddle01_flutter_client/huddle01_flutter_client.dart';

  String projectId = 'YOUR-PROJECT-ID';
  String roomId = 'YOUR-ROOM-ID';

  // Initialize your huddleClient 
  HuddleClient huddleClient = HuddleClient(projectId);
  ```
