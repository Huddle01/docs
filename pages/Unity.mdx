# Huddle01 Unity Native Plugin

The Huddle Unity Native Plugin SDK offers a streamlined solution for integrating real-time audio and video into Unity projects.
It supports multiple platforms, including Windows, macOS, Android, iOS, and WebGL.

## Pre Requisites

Before using the Huddle01 Unity SDK, ensure your setup includes the following prerequisites:

- Unity Version: 2022.3 or higher

- Unity WebRTC Package: Follow the installation guide **[here](https://docs.unity3d.com/Packages/com.unity.webrtc@3.0/manual/index.html)**.
