## Consume

### StartConsumerPeer(string peerId);

Call this function to consume individual peer. This function will invoke Event [`Huddle01Core.OnStartConsumingPeer`](/UnityWebglPlugin/methods/remotePeer#startconsumingpeereventhandlerstring-peerid) once the process is successfully done

**Parameters**
<OptionTable
  headers={["Param", "Type", "Description"]}
  options={[["peerId", "string", "The peer ID of the Remote Peer to consume from."]]}
/>

```csharp filename=".csharp"
public void ConsumePeer(string peerId)
{
    Huddle01Core.Instance.StartConsumerPeer(peerId);
}
```
