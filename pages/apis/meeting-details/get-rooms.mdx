## **Get Rooms API**

The Get Rooms API provides the ability to fetch all the rooms associated with the project.
In order to use the API, you will need to obtain an API key for the project from huddle01.dev.

## **Endpoint**

Upon success, the API will return a response with an array of rooms with the following fields:

- **`rooms`** - The list of rooms with their respective roomId, createdAt, roomLocked and metadata.
- **`pageSize`** - The number of rooms in current page
- **`current`** - The current page number
- **`nextCursor`** - The next page number
- **`prevCursor`** - The previous page number
- **`count`** - The total number of rooms in the project 

In case of an error, the API will return an error message with a corresponding status code.
